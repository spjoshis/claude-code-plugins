# Project Reporting

Create effective project reports, dashboards, and presentations that communicate status, progress, and issues to stakeholders.

## When to Use This Skill

- Weekly status updates
- Executive reporting
- Stakeholder communication
- Progress tracking
- Issue escalation
- Budget reporting
- Risk communication
- Performance measurement

## Best Practices

1. **Executive summary first** - Key points upfront
2. **RAG status** - Red/Amber/Green indicators
3. **Trend analysis** - Show progress over time
4. **Highlight exceptions** - Focus on issues and risks
5. **Data visualization** - Charts and graphs
6. **Actionable insights** - What needs decisions
7. **Consistent format** - Regular template
8. **Timely delivery** - Regular schedule
